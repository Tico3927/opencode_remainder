## 快速入门

### 注册
1. 访问 https://ai.paratera.com/#/register
2. 填写手机号等信息注册
3. 登录 https://ai.paratera.com/#/login

### 创建容器
1. 选择容器云 → 容器实例 → 创建
2. 选配：计费模式、站点、资源类型(5090)、配置、镜像、环境
3. 启动后即可用 WebSSH/SSH/Jupyter/VSCode 连接

## RTX 5090 使用指南

### 预装环境
- **CUDA**: 12.8+ （已安装）
- **NCCL**: 2.26+ （已安装）
- **Python**: 默认含 PyTorch

### PyTorch 安装
```bash
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
```

⚠️ 必须 PyTorch 2.7.0+，需要 Python 3.9+

### NCCL 修复（容器云提供预编译版本）
```bash
cp /shared-public/lib/nccl/2.26.2_cu128/libnccl.so.2 /path/to/condaenv/lib/python3.x/site-packages/nvidia/nccl/lib/
```

> 容器云已提供预编译版 nccl，无需手动编译！

### GPU Arch
- sm_120，编译程序时需修改

## SSH 获取方式

### 方式1: WebSSH（浏览器终端）
1. 容器实例列表 → 点击实例
2. 点击【SSH连接】
3. 浏览器中直接操作

### 方式2: 获取 SSH 信息（第三方工具）
1. 容器实例详情页
2. 复制 SSH 连接信息（地址+端口+用户名+密码）
3. 本地用 `ssh` / Xshell / PuTTY 连接
