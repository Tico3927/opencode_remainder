## 两种计费方式

### 按固定时长（预付费）

| 项目 | 说明 |
|------|------|
| 扣费方式 | 预付费，开机时一次性扣费 |
| 时长 | 包年(1年起) / 包月(1月起) |
| 到期策略 | 【到期自动删除】(默认) / 【到期自动续费】 |
| 自动续费 | 到期前 7 天扣费，余额不足则删除 |
| 到期处理 | 不续费 → 立即关机回收，**不可恢复** |

### 按时长（后付费）

| 项目 | 说明 |
|------|------|
| 扣费方式 | 后付费，按秒计费，按小时结算 |
| 计量 | 随时购买随时释放 |
| 回收 | 欠费自动关机 → 立即删除，**不可恢复** |

#### 注意事项
- ⚠️ **必须通过平台【关机】按钮关机**，禁止直接在 OS 关机
- 标准模式关机：云服务器不计费，云盘+网络正常计费
- 计费模式关机：所有资源继续计费
- 节省模式关机：云服务器+网络不计费，云盘正常计费
